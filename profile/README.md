# TBI AI-team

Het AI-team ondersteunt de werkmaatschappijen van TBI bij het verkennen, bouwen en opschalen van AI-initiatieven. In deze organisatie verzamelen we de centrale tooling die alle TBI-bedrijven daarbij helpt.

## Wat vind je hier?

- **[tbi-ai-backlog](https://github.com/TBI-AI-team/tbi-ai-backlog)** — de centrale backlog van AI-initiatieven uit alle werkmaatschappijen, met een boardroom-dashboard op [tbi-ai-team.github.io/tbi-ai-backlog](https://tbi-ai-team.github.io/tbi-ai-backlog/).

## Een initiatief indienen

Heb je een AI-idee voor jouw werkmaatschappij? Vul het [intake-formulier](https://github.com/TBI-AI-team/tbi-ai-backlog/issues/new?template=ai-initiatief.yml) in op de backlog. Het AI-team pakt het van daaruit op.

## Vragen?

Stel ze via een issue op de relevante repo, of mail het AI-team rechtstreeks.
